# AI Model Release Timeline

Complete timeline of major model releases, transitions, and deprecations from late 2025 through April 2026.

## 2025 Timeline

### October 2025

  - Status: Active (April 2026)

### February 2025
- **GPT-4.5** released as replacement for failed "Orion" project

## Key Trends in Timeline

### Acceleration
- Model iteration cycles shrinking (monthly variants vs yearly releases)
- Multiple variants per family (5+ GPT-5.4 versions in 3 weeks)

### Specialization
- **Thinking models** (Gemini, GPT-5.4 Thinking) dominant for reasoning
- **Specialist models** (Claude Mythos for security)
- **Lightweight variants** (Mini, Nano) for cost-efficiency

### Scale & Context
- **1M token context** now standard (Gemini, Llama 4 Scout)
- **10M token context** achieved (Llama 4 Maverick)
- Enables full codebase/video ingestion

### Open-Source Maturity
- **Llama 4** competitive with proprietary models at fraction of cost
- Open-weight models now viable for enterprise workloads
